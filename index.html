Epidemics on Networks

This project provides tools that are useful for the simulation and modeling of epidemic spread on networks. Most of these are described in the forthcoming book Mathematics of Epidemics on Networks: From Exact to Approximate Models by Kiss, Miller, and Simon.

At present the project is limited to a (nearly) complete python module which provides tools for simulating epidemics in networks and for solving the many systems of differential equations described in the text. As the project progresses we anticipate adding better visualization tools and including other languages.

We encourage other researchers to become involved by using the code, suggesting improvements, and even making additions.
